### 1️⃣ 快速了解项目

## 📈 项目历程

| 时间 | 阶段 | 主要成果 |
|------|------|---------|
| 2025年3-5月 | 基础功能期 | MSD标签打印、品质异常、IQC检验 |
| 2025年6月 | 生产信息化期 | 工单报工、SOP线上化、物料&BOM |
| 2025年7月 | 测试系统化期 | 测试任务单、数据看板 |
| 2025年8月 | 销售&PMC期 | 销售订单、工单拆分、送货/退货单 |
| 2025年9月 | 软件开发期 | 日志管理软件、软件授权验证 |
| 2025年10月 | 主系统搭建期 | 销售模块闭环、打样需求管理 |
| 2025年11月 | 生产核心期 | 生产计划、MRP运算、库存、采购 |
| 2025年12月 | 试运行期 | 新工单试运行、成本机制 |
| 2026年1月 | 正式上线期 | 财务模块、RMA售后、正式上线 |
