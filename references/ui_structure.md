# 简道云智能助手界面结构参考

## 导航路径

```
简道云 → 应用（ERP-零创）→ 目标表单 → 表单设计
  → 顶部标签「扩展功能」→ 左侧菜单「智能助手」
```

## 智能助手列表页面结构

### 页面布局
- **左侧**: 智能助手名称列表（可滚动）
- **右侧**: 选中助手的详细配置工作流画布

### 列表项信息
每条智能助手列表项显示：
1. 助手名称
2. 触发来源表单名
3. 触发动作类型（新增数据/修改数据/删除数据/流程状态变更）
4. 状态开关（启用/停用）

## 工作流画布节点类型

### 节点图标识别

| 类型 | 外观特征 | 典型位置 |
|------|---------|---------|
| 表单触发 | 圆角矩形，蓝色背景 | 工作流最顶端 |
| 计算节点 | 圆角矩形，蓝色背景 | 公式运算 |
| 查询节点 | 圆角矩形，蓝色背景 | 获取外部数据 |
| 修改数据 | 圆角矩形，蓝色背景 | 写入目标表单 |
| 新增&修改 | 圆角矩形，蓝色背景 | 条件性写入 |
| 分支条件 | 菱形，白色背景 | 条件判断分流 |
| AI处理 | 圆角矩形，蓝色背景+AI标识 | AI调用 |
| 发送消息 | 圆角矩形，蓝色背景 | 企业微信等通知 |

### 连接线
- **实线**: 正常流转
- **虚线**: 条件分支/异常路径

## 助手详情面板信息

点击单个智能助手后，右侧显示：

### 1. 基本信息
- 助手名称（可编辑）
- 创建时间
- 最后修改时间
- 执行记录（最近执行次数）

### 2. 触发配置
```
触发方式:
├── 触发来源: [下拉选择表单]
├── 触发动作: 
│   ├── 当有新增数据时
│   ├── 当数据被修改时（可选指定字段）
│   ├── 当数据被删除时
│   └── 当流程状态变更时（可选指定状态）
└── 触发条件: [公式表达式，可选]
```

### 3. 工作流画布
- 可缩放（鼠标滚轮）
- 可拖拽移动画布
- 节点可点击查看详细配置
- 长工作流需向下滚动查看

## 截图建议区域

### macOS Chrome 浏览器窗口布局
```
┌──────────────────────────────────────────────────────┐
│  标签栏 / 地址栏                                     │
├──────────┬───────────────────────────────────────────┤
│          │                                           │
│  左侧     │         工作流画布区域                     │
│  助手     │                                           │
│  列表     │    ┌─────────┐                            │
│  (~300px) │    │ 触发节点  │──→ [计算]──→ [查询]        │
│          │    └─────────┘       ↓                    │
│          │                  ┌──────┐                 │
│          │                  │ 分支  │──→ [分支A]      │
│          │                  └──────┘    ↓            │
│          │                           [修改数据]       │
│          │                                           │
├──────────┴───────────────────────────────────────────┤
│  底部状态栏 / Dock                                   │
└──────────────────────────────────────────────────────┘
```

建议截图时：
1. 最大化浏览器窗口以获取最大可视面积
2. 使用 Cmd+ 缩放至合适比例（推荐80%-100%）
3. 如工作流过长，分段截取后手动拼接
